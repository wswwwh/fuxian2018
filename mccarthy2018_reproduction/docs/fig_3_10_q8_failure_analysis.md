This note audits only the q=8 period-q halo example used by Fig. 3.10.

The per-segment endpoint errors are all below `4.45e-15` for the 15 patch
continuity checks, with a final terminal symmetry endpoint error of
`6.14e-16`.

## Cause classification

**A. Multiple-shooting solution itself does not satisfy full periodicity:** no.
The half-period patch chain is internally consistent. Patch-to-patch
continuity and terminal symmetry are at roundoff-scale residuals.

**B. Single-arc propagation diverges on a highly unstable orbit:** yes. The
full-period single-shoot closure is `3.91`, while the patch residuals are
`1e-14` scale. The max monodromy multiplier magnitude is `3.43e16`, so a
single integration from the first state is extremely sensitive to tiny local
errors even though Jacobi drift remains small.

**C. Period or segment-duration definition is wrong:** no evidence. The audit
uses `segment_duration * segments == half_period` and `period == 2 *
half_period`; the segment-duration consistency error is exactly zero in the
stored CSV.

**D. Symmetry reflection or trajectory sampling closure check is wrong:** no
evidence for the plotted patch-reflected trajectory. Sampling closure is zero
by construction because the second half is generated by symmetry reflection.
The independent half-period single-arc symmetry residual is not zero
(`1.65e-04`), but that is the same instability issue as the full-period
single-arc check.

**E. Other cause:** numerical ill-conditioning is present in the q=8 monodromy
computation because the orbit is extremely unstable. This affects single-arc
verification, but it does not invalidate the patch-local multiple-shooting
continuity check.

## Conclusion

The q=8 panel should remain classified as `shape-match with local numerical
overlay`. The available evidence supports a local multiple-shooting
approximation with consistent period, patch, and symmetry definitions, not a
robust full-period single-shooting numerical reproduction.
